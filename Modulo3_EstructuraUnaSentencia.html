ESTRUCTURA DE UNA SENTENCIA EN CSS


A continuación se presenta un ejemplo de la estructura que un selector de CSS tiene :





El selector apunta al elemento HTML al que desea aplicar estilo.

El bloque de declaración (declaration) contiene una o más declaraciones separadas por punto y coma.

Cada declaración incluye un nombre de propiedad CSS (property) y un valor, separados por dos puntos.

Varias declaraciones CSS se separan con punto y coma y los bloques de declaración están rodeados por llaves.

Ejemplo explicado:

  • p es un selector en CSS (apunta al elemento HTML que desea aplicar estilo: <p>).
  • color es una propiedad y red es el valor de la propiedad
  • text-alignes una propiedad y centeres el valor de la propiedad
