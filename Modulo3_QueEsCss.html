¿QUÉ ES CSS?


  • Las letras CSS vienen de Cascading Style Sheets (Hojas de estilo en cascada).
  • CSS se encarga de describir como se mostrarán los elementos HTML en la pantalla.
  • CSS ahorra mucho trabajo. Puede controlar el diseño de varias páginas web a la vez.
  • Las hojas de estilo externas se almacenan en archivos CSS
